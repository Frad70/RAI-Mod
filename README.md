# RAI Mod (NeoForge 1.21.1)

Серверный мод, который создаёт «симулированных игроков» с собственной памятью, целями и долгосрочным прогрессом.

## Что уже реализовано в этом коммите

- Серверный конфиг с hot-reload и безопасными лимитами.
- `AIDirectorService`: центральный оркестратор ИИ-ботов.
- Перезапуск runtime при изменении конфига без потери сущностей в persistence-слое.
- Система поведения через приоритизируемые цели:
  - защита базы,
  - возврат лута с трупа,
  - рейд,
  - прогрессия/фарм.
- Память бота:
  - точки знаний мира,
  - приоритеты для рейдов,
  - боевой журнал.
- Каркас интеграций под TACZ / Corpse / SimpleVoiceChat / SecurityCraft / PhysicalStats.

## Конфиг по умолчанию

- `maxSimulatedPlayers = 5` (жёсткий потолок 20)
- `minActiveChunks = 1` (жёстко зафиксировано)
- `maxActiveChunks = 4` (жёсткий потолок 12)
- динамический бюджет чанков включён
